# CamelusMips

![Camel](doc/camel.jpg)

Yet Another MIPS Emulator.

## Instructions Supported

- R-type: addu, add, subu, sub, and, or, xor, nor, sltu, slt, sllv, srlv, srav, jr, sll, srl, sra
- I-type: addiu, addi, andi, ori, xori, lui, lw, sw, beq, bne, sltiu, slti
- J-type: j, jal
